Tarea #1
Objetivo:
Que el estudiante utilice colecciones de java existentes y no desarrollarlos desde cero, entendiendo el tema de reutilización de componentes.
Explicación:
El objeto ArrayList es utilizado para construir arreglos dinámicos, es decir, que no tienen un tamaño fijo. Este objeto permite realizar operaciones comunes como agregar, eliminar, buscar y modificar.
Desarrollo:
Se deberá crear una aplicación Java, que permita ingresar, modificar, eliminar, buscar y mostrar datos utilizando el objeto ArrayList. Los datos serán de tipo String.
Opciones del menú y consideraciones:
	1. Ingresar: el usuario digita el nuevo dato.
	2.Modificar: el usuario digita el dato a modificar y el nuevo dato por el que se va a sustituir. Debe realizarse una búsqueda del dato a modificar y mostrar un mensaje indicando en caso de no encontrarlo.
	3.Eliminar: el usuario digita el dato a eliminar. Debe realizarse una búsqueda del dato a eliminar y mostrar un mensaje indicando en caso de no encontrarlo.
4.Buscar: el usuario digita el dato a buscar y la aplicación le muestra la posición en que se encuentra. Si no se encuentra el dato se mostrará un mensaje indicándolo.
5.Mostrar: la aplicación recorre todos los valores y muestra la posición y el valor de cada elemento de la lista.
6.Salir: fin de la aplicación.
Deben incluirse validacion es para que el programa nunca dé errores numéricos ni de otro tipo.
